Design Overview

The Phones-R-Us project required the development of two precision casings for their modular mobile device. The bottom casing was modelled directly from a detailed client drawing provided by Phones-R-Us, which included all necessary dimensions, tolerances, and geometric specifications. This drawing was clear and well-structured, allowing for straightforward translation into CAD with minimal interpretation required.

The top casing presented a different challenge, as it required complete reverse engineering from a physical sample part. Unlike the bottom casing, no formal drawing existed for the top casing initially. A reference drawing was provided, but it had significant limitations that required careful analysis and problem-solving to overcome. Both components were designed in Autodesk Fusion 360, a professional CAD software that enabled precise modelling of complex geometries, including the curved surfaces and non-uniform depth of the top casing.

The design process involved multiple iterations to ensure proper fit between the two casings, as well as compatibility with existing components from other suppliers. The final designs were exported as STL files, the industry-standard format for additive manufacturing, ready for the 3D printing phase of the project.

⚠️ Drawing Challenges: The top casing drawing provided by Phones-R-Us had multiple missing dimensions and confusing cross-sections that made accurate interpretation difficult. A key issue was that it was often unclear where exactly on the top casing each measurement applied, as the drawing lacked clear reference points and datum indicators. Some critical dimensions were completely absent from the drawing, while others were placed ambiguously on the drawing sheet with unclear leaders pointing to uncertain locations. The cross-sections provided did not always align with the main views, creating further confusion about which features corresponded to which measurements.
✓ Solution Approach: Missing values were systematically calculated by cross-referencing other known measurements on the drawing and using geometric relationships between features. Multiple views (front, top, side, and isometric) were compared to determine the correct locations for each ambiguous dimension. Logical assumptions had to be made where information was incomplete, and these assumptions were validated through test prints and fit checks with the bottom casing. This methodical approach resulted in an accurate CAD model despite the drawing's limitations.

⚠️ Drawing Challenges: The bottom casing drawing presented a different type of challenge. While the drawing itself was clear and complete, the challenge was ensuring that the modelled geometry would properly interface with the top casing and with components from other suppliers. The drawing specified standard tolerances of ±0.1mm, but achieving this consistently across multiple prints required careful consideration of the FDM process capabilities. Additionally, some internal features required supports during printing, which affected surface finish in critical mating areas.
✓ Solution Approach: The drawing was first translated directly into CAD with high fidelity to the dimensions. For internal features requiring supports, the orientation of the part during printing was optimized to minimize support material in critical areas. Post-processing procedures were developed to clean support interfaces without affecting dimensional accuracy. The model was then validated against the drawing at each stage, and test prints were used to verify fit with the top casing before proceeding to full production.

Fusion 360 3D Models

Top Casing - 3D Model

The top casing 3D model was built in Fusion 360 using the reverse engineered dimensions. Key challenges included modelling the non-flat geometry with varying depth across the part, creating smooth transitions between different depth zones, and designing snap-fit features that would engage reliably with the bottom casing. The model required three iterations to achieve proper fit.

Modelling Challenge: The top casing had curved surfaces that were difficult to measure accurately from the physical sample. The drawing lacked curvature radii information.
Solution: Curvature was approximated using spline profiles in Fusion 360, then refined through test prints until the fit with the bottom casing was seamless.
STL Exported

Bottom Casing - 3D Model

The bottom casing 3D model was developed directly from the client drawing. The consistent depth of 13mm made modelling straightforward. Key features included the battery compartment, mounting bosses, alignment ribs, and the outer profile. The model was verified against the drawing before STL export.

Modelling Challenge: Some internal corners were specified with sharp radii that would be difficult to achieve with FDM printing without supports.
Solution: Sharp internal corners were redesigned with 1mm fillets after confirming with the client that this would not affect component fit. This improved printability without compromising function.
